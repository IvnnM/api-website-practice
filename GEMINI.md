# Gemini

## Overview

This project is about Front-End development using an API.

## Core Technologies

- HTML
- Tailwind CSS
- JavaScript
- Axios

## Coding Style

- Use 2 spaces for indention.
- Naming is always camel case.

## Rules

- The output code must be **minimalist and immediately functional**. Do not include advanced error handling, logging, or extensive comments unless explicitly requested. Use direct, imperative logic.
